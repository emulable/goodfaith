## Notes on Compression

The framework has a micro edition for constrained contexts (Copilot, custom agents with character limits). Building it taught hard lessons.

**Characters â‰  bytes â‰  tokens.** This confusion derails every compression attempt. Chinese characters are ~3 bytes each in UTF-8. An 8000-character limit means ~22-24KB in bytes. `wc -c` counts bytes. `wc -m` counts characters (locale-dependent). Python `len(content)` is most reliable for characters. **Chinese saves characters, not tokens or bytes.** Chinese is semantically denserâ€”one character carries more meaning than one English letter. **Start low, build up.** The instinct is to write everything, then trim. This creates oscillationâ€”trim too much, add back, trim again, forever. **70% priming, 30% reference.** The micro should mostly install operational capacityâ€”momentum briefing, substrate correction, anticipatory markers.
